# qtcloud-auth · 量潮身份云

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)

量潮身份云（qtcloud-auth）是量潮科技旗下的**身份认证与权限管理云服务**，提供企业级的统一身份管理、认证授权、安全策略等核心能力。

## 产品概述

- **统一身份管理**：用户、组织、角色的创建、同步与生命周期管理
- **认证服务**：OAuth 2.0 / OIDC / SAML / LDAP 等标准协议支持
- **授权与权限**：RBAC、ABAC、ReBAC 等权限模型
- **安全策略**：多因素认证（MFA）、会话管理、令牌管理

## 模块

| 路径 | 说明 |
|------|------|
| `apps/` | 应用服务（含领域子模块） |
| `src/provider/` | 认证提供者实现 |
| `docs/` | 产品文档 |
